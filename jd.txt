1. Position Overview
The Associate Data Scientist & Machine Learning Engineer plays a vital role in transforming complex transactional,
behavioral, and operational data into production-ready analytical pipelines and predictive frameworks. Working at the
intersection of statistical learning and robust web technologies, the successful candidate will not only build highly
accurate machine learning models but also assist in exposing these solutions via microservices and interactive
analytical interfaces.
"This role is uniquely tailored for a highly analytical engineer who possesses strong foundations in statistical predictive
models (such as Gradient Boosting and Regression) alongside full-stack engineering capability to build, serve, and
visualize data applications."
2. Key Responsibilities & Duties
Predictive Modeling & Forecasting: Design, train, evaluate, and optimize regression, classification, and timeseries forecasting models (e.g., XGBoost, Random Forests, LightGBM) on large-scale datasets to identify patterns
and forecast operational metrics.
End-to-End Pipeline Engineering: Build clean ETL (Extract, Transform, Load) pipelines utilizing Python (Pandas,
NumPy) and SQL to scrub, normalize, and structurally enrich raw data from relational databases and cloud sources.
BI & Dashboard Automation: Design and automate intuitive corporate dashboards using tools like Power BI and
Excel to communicate daily, weekly, or monthly performance metrics clearly to senior management.
API & Service Development: Package trained machine learning algorithms into fast, scalable REST APIs (using
FastAPI or Flask) and build dynamic frontend layers (using React) to allow internal applications or external systems
to consume predictive insights.
NLP & Text Extraction: Implement Natural Language Processing (NLP) techniques utilizing packages such as
spaCy to parse, categorize, and score unstructured text profiles (e.g., resume matching, customer feedback
sentiment analysis).
